# INSTALL

## LARAVEL

```bash
composer create-project laravel/laravel rub-fii-investor
npm install
npm run dev
npm run build
php artisan migrate
```

## FIRST COMMIT

```bash
git init
git branch -M main
git remote add origin https://github.com/ricardoub/rub-fii-investor.git
git add .
git commit -m "FIRST COMMIT: laravel install"
git push -u origin main
```
